# Pencil-Sketcher
A simple pencil sketcher using Opencv and Python

## Try Demo!
No need to install try it out. https://pencilsketchers.herokuapp.com/

## Requirements
- numpy==1.22.4
- opencv-python-headless
- Pillow==9.1.1
- streamlit==1.10.0

## Installation

Must have to satisfy all the requirements

```bash
  streamlit run pencil_sketch.py
```
